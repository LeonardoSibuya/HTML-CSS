ANOTAÇÕES 2 HTML (13/10)



H1: serve para titulos e subtitulos - vai até h6


P: paragrafo


BR: pular linha 


Link: digitar acima do title, escrever favicon para gerar icon 


Strong: tag para deixar em negrito / destaque


Em: Usar para deixar em italico


Mark *: Cria um marca texto, porém não há a tag no code, então temos que digitar CTRL + SHIFT + P, digitar ABB e escrever MARK


Small: texto pequeno


Del: texto riscado


Ins: testo sublinhado


Sub: Texto subescrito


Sup: Texto sobrescrito


Code *: Fonte espaçada


Pre: Identa como programação, quebra linha


q: Aspas


Blockquote: citações, texto fica mais em baixo e direita.


Abbr title= : explica o que seria aquela abreviação


ol: Listas ordenadas


Li: colocar dentro de ol para organizar a lista


ul: Lista não ordenada


type: dentro de ol ou ul vc escolhe o tipo da lista


start: voce escolhe por qual letra ou numero vai começar, mas tem que ser em ordem numerica


Segurar ALT: Seleciona mais de uma vez


Dl: Lista de definição sem formatação


Dt: termo


Dd: descrição


-----------------------------------------------------------

Semantica: é importante o significado e não a forma, exemplo, não utilizar bg collor para mudar a cor da pagina, pois hoje em dia é utilizado css para folha de estilo
